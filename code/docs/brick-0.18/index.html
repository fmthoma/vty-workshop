brick-0.18: A declarative terminal user interface library

Write terminal applications painlessly with brick! You write an event handler and a drawing function and the library does the rest.

module Main where

import Brick

ui :: Widget ()
ui = str "Hello, world!"

main :: IO ()
main = simpleMain ui
